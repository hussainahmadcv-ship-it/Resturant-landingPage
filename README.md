# Gourmet Haven - Restaurant Landing Page

## Overview
**Gourmet Haven** is a responsive restaurant landing page built with **HTML**, **Tailwind CSS**, and **JavaScript**. It includes:

- Hero section  
- About & Story  
- Menu preview  
- Features section  
- Contact & reservation form  
- Footer with social links  

> **Disclaimer:** This website is a **demo project**. All images are from [Unsplash](https://unsplash.com/) and all content is placeholder. This is not a real restaurant website.
